# Quantumult-X 什锦配置

本配置根据神机规则以及小白配置修改而来，集合了各位大佬的规则·复写·脚本，包括神机规则·野比部分脚本等。
一份懒人配置，纯粹自用。



### 使用自定义 GeoIP 文件
建议在「其他设置」里「GeoLite2」的「来源」填写使用「 https://raw.githubusercontent.com/Loyalsoldier/geoip/release/Country.mmdb 」 并开启「自动更新
